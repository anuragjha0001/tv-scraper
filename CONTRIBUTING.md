# Contributing to TV Scraper

## Setup

Clone the repository:
git clone https://github.com/anuragjha0001/tv-scraper-py.git
cd tv-scraper

Install in development mode:
pip install -e ".[dev]"

## Tests

Run tests:
pytest tests/ -v

## Questions?

Open an issue: https://github.com/anuragjha0001/tv-scraper-py/issues
